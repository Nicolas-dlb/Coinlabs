## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Application de trading permettant aux utilisateurs d'analyser les gains et pertes provoqués par leurs placements grâce à la disposition d'un portefeuille fictif : https://coinlabs.web.app

### Built With

* React
* Redux
* Typescript
* Jest/Enzyme
* Firebase

<!-- GETTING STARTED -->
## Getting Started

To run the project follow these simple steps.


### Installation
 
1. Clone the repo
```sh
git clone https://github.com/Nicolas-dlb/Coinlabs
```
2. install all dependances
```sh
npm install
```
3. run the project
```sh
npm start
```

### Tests

run unit tests
```sh
npm test
```
